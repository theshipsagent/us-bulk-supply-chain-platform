## PAYWALLED ARTICLES - STRATEGIES TO ACCESS

### Strategy 1: ResearchGate Author Request (RECOMMENDED - FASTEST)

**Wetzstein et al. (2021) "Transportation costs: Mississippi River barge rates"**

1. Go to: https://www.researchgate.net/
2. Search for: "Brian Wetzstein" + "Transportation costs Mississippi River"
3. Locate article in profile or search results
4. Click "Request PDF" button
5. **Timeline:** Author usually responds within 24-48 hours

**Alternative:** Search the article title directly in ResearchGate's main search

---

### Strategy 2: University Library Proxy Access

**If you have university affiliation:**

1. Find university library website
2. Access journal through institutional subscription
3. **ScienceDirect Example:**
   - Use library proxy URL: `http://ezproxy.[yourschool].edu/login?url=https://www.sciencedirect.com`
   - Search for Journal of Commodity Markets, Vol 21

**Specific Libraries with Strong Access:**
- Cornell University Library (agricultural focus)
- University of Wisconsin-Madison (transportation & ag economics)
- Purdue University Library (author's institution - likely has access)

---

### Strategy 3: Email Authors Directly for Preprints

**Author Contact:**

**Brian Wetzstein**
- Current affiliation: Purdue University, Department of Agricultural Economics
- Institution page: https://ag.purdue.edu/agecon/
- Typical format: `bwetzstein@purdue.edu` or similar
- **Email template:**

```
Subject: Preprint Request - Transportation costs: Mississippi River barge rates

Dear Dr. Wetzstein,

I am researching inland waterway transportation economics and 
would appreciate a copy of your 2021 article "Transportation costs: 
Mississippi River barge rates" published in the Journal of Commodity Markets.

[Your Name]
```

**Success Rate:** ~70-80% (academics generally willing to share)

---

### Strategy 4: Interlibrary Loan (ILL)

**If not affiliated with university:**

1. Check if public library offers ILL services
2. Request through library system to:
   - Local university library
   - Research library consortia (e.g., Center for Research Libraries)
3. **Cost:** Usually free or $5-10 per article
4. **Timeline:** 3-7 business days
5. **Popular ILL platforms:**
   - RapidILL (inter-library loan network)
   - OCLC WorldCat (library search tool)

---

### Strategy 5: Purchase Individual Article

**Last Resort Options:**

- **ScienceDirect:** $31-40 per article
- **Wiley Online Library:** $30-35 per article
- **JSTOR:** $19-30 per article (if institutional access unavailable)

## ACCESSING RESEARCH DATABASES

### USDA Data Portals (All Free)

**1. Agricultural Marketing Service (AMS) - Grain Transportation**
- URL: https://www.ams.usda.gov/services/transportation-analysis
- Data: Weekly barge rates by segment, railcar offers, futures prices
- Format: Excel download, web dashboard
- Frequency: Updated Thursdays

**2. Open Ag Transport Data (Socrata SODA API)**
- URL: https://agtransport.usda.gov/
- Format: REST API + dashboard
- Data: Barge movements, rates, commodity flows
- For programmers: Use SODA API for automated data pulls

### Army Corps of Engineers (USACE)

**1. Institute for Water Resources (IWR)**
- URL: https://www.iwr.usace.army.mil/
- Data: Waterborne Commerce of the United States annual reports
- Frequency: Annual (some monthly data)
- Format: CSV, Excel, PDF

**2. USACE Geospatial Open Data Portal**
- URL: https://geospatial-usace.opendata.arcgis.com/
- Format: Shapefile, GeoJSON, REST API
- Data: Navigation channels, locks, ports, waterway network
- No login required

**3. Navigation & Civil Works Decision Support Center (NDC)**
- URL: https://www.iwr.usace.army.mil/About/Technical-Centers/NDC-Navigation-and-Civil-Works-Decision-Support/
- Contact: Request specific datasets or monthly lock operation data

### Bureau of Transportation Statistics (BTS)

**1. Freight Analysis Framework (FAF)**
- URL: https://faf.ornl.gov/
- Download: https://faf.ornl.gov/faf5/data/download_files/
- Format: CSV, Access database
- Data: Regional freight flows, commodity-mode distributions
- License: Public domain

**2. National Transportation Atlas Database (NTAD)**
- URL: https://geodata.bts.gov/
- Format: REST API, Shapefile, GeoJSON
- Data: Rail, highway, waterway networks
- Tools: Interactive explorer, direct download

**3. Transportation Services Index (FRED)**
- URL: https://fred.stlouisfed.org/series/WATERBORNE
- Series ID: WATERBORNE (monthly inland waterborne tonnage)
- Format: CSV download, interactive chart
- Data: 2000-present, seasonally adjusted & unadjusted

## TROUBLESHOOTING ACCESS ISSUES

### "Paywall/Login Required" on ScienceDirect?
1. Try ResearchGate author request (usually faster)
2. Check if your public library has journal access
3. Contact author directly for preprint

### Dissertation PDF Won't Download?
1. Check file size (Purdue often offers full PDF)
2. Try alternative format if available (some sites offer ePub, Kindle)
3. Contact university directly for alternative delivery

### Data Portal Site Down?
- Keep backup URLs from Archive.org: https://archive.org/
- Search "Wayback Machine" + old URL to access cached version
- Contact issuing agency for data availability

### API Rate Limits Exceeded?
- USDA/BTS APIs are generous; unlikely to hit limits for research
- If needed, cache data locally and update periodically
- Consider downloading full datasets instead of API calls
