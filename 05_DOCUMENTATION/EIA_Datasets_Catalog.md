## Overview

This comprehensive catalog contains all 85 datasets published by the U.S. Energy Information Administration (EIA) on data.gov. The datasets cover a wide range of energy topics including electricity, petroleum, natural gas, coal, renewable energy, and international energy data.

### Dataset Format Distribution
- **API:** 33 datasets
- **Excel/XLS:** 12 datasets
- **HTML:** 8 datasets
- **CSV:** 3 datasets
- **JSON:** 1 dataset
- **PDF:** 1 dataset
- **SHP (Geospatial):** 1 dataset

### Major Topic Categories
- **Climate & Emissions:** 11 datasets
- **Energy Infrastructure:** 3 datasets
- **Energy Resources:** 3 datasets
- **Energy Demand:** 1 dataset
- **Energy Supply:** 1 dataset

### Common Tags (Most Frequent)
- **api:** 32 datasets
- **crude-oil:** 12 datasets

- **projections:** 10 datasets

## Data Access Notes

### API Access
Most datasets marked with "API" format require an API key from EIA. You can register for a free API key at: https://www.eia.gov/opendata/register.php

### Bulk Downloads
The Bulk Download Facility (Dataset #14) provides access to entire datasets in single ZIP files, which can be more efficient for large-scale data retrieval.

### Data Formats
- **API**: Machine-readable JSON format via RESTful API
- **XLS/EXCEL**: Microsoft Excel spreadsheet files
- **HTML**: Web-based interactive data browsers
- **CSV**: Comma-separated values files for easy import
- **SHP**: Shapefile format for geospatial data (GIS compatible)
- **JSON**: JavaScript Object Notation
- **PDF**: Portable Document Format
