## GenealogyBank Search Strategy

### Phase 1: Broad Exploration (Day 1-2)

**Search #1: Agency Fee Notices**
```
Search terms: "agency fee" OR "agent fee" OR "agent's fee"
Publication: Journal of Commerce (New York)
Date range: 1915-1920
```

**Search #2: Port Charges Announcements**
```
Search terms: "port charges" AND "agent"
Publication: Journal of Commerce
Date range: 1915-1920
```

**Search #3: Disbursement Rates**
```
Search terms: "disbursement" AND "vessel" OR "ship"
Publication: Journal of Commerce
Date range: 1915-1920
```

**Search #4: Agent Commission**
```
Search terms: "agent commission" OR "agent's commission"
Publication: Journal of Commerce
Date range: 1915-1920
```

### Phase 2: Benchmark Year Deep Dives (Day 3-7)

Repeat searches for each benchmark period:

**1915-1920 (WWI Era)**
- Focus on: War risk fees, government controls, Shipping Board impacts
- Look for: Rate notices, tariff changes, agent compensation discussions

**1935-1940 (Depression Era)**
```
Date range: 1935-1940
Same search terms as above
Additional: "maritime commission" AND "rates"
```

**1950-1960 (Post-WWII)**
```
Date range: 1950-1960
Same search terms
Additional: "federal maritime board" AND "agent"
```

**1970-1980 (Container Revolution)**
```
Date range: 1970-1980
Search terms: "container" AND "agent fee" OR "agency charges"
Look for: How containerization impacted agency compensation
```

**1990-1995 (Pre-Internet)**
```
Date range: 1990-1995
Search terms: Same as above
Additional: "port agent" AND "competition"
```

### Phase 3: Specific Port Searches (Day 8-10)

Target major ports mentioned in USACE data:

```
"New York" AND ("agency fee" OR "port agent" OR "disbursement")
"Norfolk" AND ("agency fee" OR "port agent")
"Baltimore" AND ("agency fee" OR "port agent")
"New Orleans" AND ("agency fee" OR "port agent")
"Houston" AND ("agency fee" OR "port agent")
"Long Beach" AND ("agency fee" OR "port agent")
"Los Angeles" AND ("agency fee" OR "port agent")
```

### Phase 4: Alternative Publications (Day 11-15)

If Journal of Commerce yields limited results, try:

**Marine Journal**
**Shipping Illustrated**
**Pacific Marine Review** (for West Coast)
**New York Times** (business section)
**Wall Street Journal** (shipping news)

### Data Extraction Protocol

For each relevant article found:

1. **Save immediately:** Download PDF or screenshot
2. **Document citation:**
   - Newspaper name
   - Date (MM/DD/YYYY)
   - Page number
   - Article title
   - GenealogyBank URL

3. **Extract data:**
   - Fee amount mentioned
   - Port location
   - Vessel type (if specified)
   - Fee structure (flat, %, per ton, etc.)
   - Context (rate increase, new tariff, dispute, etc.)

4. **Organize by year:**
   - Create folder structure: 1915-1920, 1935-1940, etc.
   - Name files: YYYYMMDD_Publication_Topic.pdf

### Expected Findings

**Likely to find:**
- Rate increase announcements
- Tariff notices
- Industry dispute coverage
- Conference agreement changes
- Regulatory action reports

**Unlikely to find:**
- Actual fee schedules (commercially sensitive)
- Detailed breakdowns (proprietary)
- Small port specific rates (focused on major ports)

**Realistic expectation:** 10-30 data points across 80 years

### Subscription Risk:
**Problem:** Forget to cancel, charged for additional months
**Mitigation:**

- Set phone alarm
- Add to task list
- Tell someone to remind you

**Problem:** Lose access to bookmarked articles after cancellation
**Mitigation:**

- Take screenshots
- Export citations to external spreadsheet
- Never rely on in-platform bookmarks

**Problem:** Subscribe but find minimal useful data
**Mitigation:**
- Complete Tier 1 (free sources) first
- Assess likelihood before subscribing
