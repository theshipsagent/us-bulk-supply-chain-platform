8-Phase Commodity Classification System

Phase Function Output
0: Preprocessing Load, validate, normalize Clean manifest data
1: White Noise Filtering Remove non-cargo shipments Cargo-only records
2: Carrier Exclusions Exclude container carriers Bulk cargo focus
3: Carrier Classification Vessel type detection Bulk, tank, general cargo
4: Main Classification Commodity keyword matching Cement, steel, grain, etc.
5: HTS4 Alignment HS code cross-validation Confidence scoring
6: Final Catchall Unclassified resolution Complete coverage
7: Enrichment Consignee matching, attributes Analysis-ready dataset

Integration Capabilities

Master CLI

# Data operations
report-platform data download --source epa_frs
report-platform data status

# Toolset operations
report-platform barge-cost \
  --origin "Houston" \
  --destination "Memphis" \
  --commodity cement

report-platform rail-cost \
  --origin "Baton Rouge" \
  --destination "Chicago"

report-platform facility-search \
  --state LA \
  --naics 327310 \
  --radius 50

# Vessel intelligence
report-platform vessel classify \
  --input panjiva.csv

# Report generation
report-platform report generate \
  --report cement_commodity \
  --format docx
