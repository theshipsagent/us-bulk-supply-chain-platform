# Knowledge Bank — Cross-Cutting Intelligence Layer

**Purpose**: Centralized repository for reference data, competitive intelligence, facility registry, and reusable analytical patterns used across all toolsets and commodity modules.

---

## WHAT IS THE KNOWLEDGE BANK?

The Knowledge Bank is a **single source of truth** for:

1. **Reference Data** - Shared taxonomies, lookups, and classification systems (NAICS, HTS, port codes, etc.)
2. **Master Facility Register** - Unified facility profiles linking fragmented data sources
3. **Competitive Intelligence** - Company profiles, ownership trees, market share analysis
4. **Analytical Patterns** - Reusable frameworks for supply/demand, trade flow, transport cost analysis
5. **Market Intelligence Commons** - Cross-commodity research (policy briefs, economic reports, technology trends)
6. **Client Intelligence** - Client-specific context for segmented reporting (SESCO, future clients)
