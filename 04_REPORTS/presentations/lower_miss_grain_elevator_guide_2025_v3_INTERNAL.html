1. Data Sources

Southport Agencies -- Vessel Lineup Reports
PRIMARY

Daily vessel lineup sheets covering Miss. River, Texas, and PNW terminal systems. Source: /Users/wsd/Projects/southport-agencies/exposed-files/excel/ (945 Excel files, 2.1 GB). Table: grain_southport_lineup in data/analytics.duckdb. 15,569 rows for MISS_RIVER 2025; 255 report days; 12 terminals. Schema: vessel_name, terminal_name, status_type (ETA/FILING/LOADING/SAILED), report_date, mt_numeric, commodity_code, destination, delay_days_min, delay_days_max.

~40--70% NULL mt_numeric for LOADING records is a structural artifact -- many vessels file "RVT" (quantity to be determined) until loading commences. FGIS certified tonnage is the authoritative volume measure.
USDA FGIS -- Federal Grain Inspection Service Export Certificates
PRIMARY

Official grain inspection certificates for all US grain exports. Table: grain_fgis_certs -- 2,207 certificates for port='MISSISSIPPI R.' in 2025 (Jan--Sep 2025). Total certified volume: 64.5M MT. Schema: serial_no, cert_date, vessel, grain, grade, pounds, metric_ton, destination, port, test_weight, moisture_avg, fm_pct, protein_avg. Data coverage: Jan 1 -- Sep 25, 2025.

2. Stow Factor Methodology (v3 New)

Stow factors computed from USDA FGIS certified test weights per grain grade:

FormulaDescription
SF (ft³/LT) = 2788.08 / TW 2788.08 = 2240 lb/LT × 1.2445 ft³/bu (standard bushel volume)
SF (m³/MT) = (2788.08 / TW) × 0.028317 / 1.01605 Convert ft³/LT to m³/MT (1 ft³ = 0.028317 m³, 1 LT = 1.01605 MT)
SF Untrimmed = SF (ft³/LT) × 1.06 6% allowance for grain surface not leveled (standard bulk carrier practice)

Reference: Isbester, Bulk Carrier Practice Ch.5; Meurn & Sauerbier, Stowage. Grain stowage factors CSV: 02_TOOLSETS/vessel_voyage_analysis/grain_stowage_factors.csv

Color coding in report: Orange highlight = light grain >52 ft³/LT (typically wheat, sorghum); Blue highlight = heavy corn <47 ft³/LT. No highlight = typical 47--52 ft³/LT range.

3. Metric Methodology

Queue Wait Time

ETA first appearance date per vessel--terminal pair to first LOADING date, filtered to <= 30 days. Metric: MEDIAN. SQL: DATEDIFF('day', first_eta, first_loading) <= 30.

Loading Duration

Count of distinct LOADING report dates per vessel--terminal pair per year. Represents the number of daily lineup reports showing that vessel as LOADING. Metric: MEDIAN loading days per vessel visit.

Anchorage Wait Time (v3 corrected)

From MRTIS event_log: DurationToNextEventHours for events where ZoneType='ANCHORAGE' OR Zone LIKE '%Anch%', filtered to < 500h. Aggregated by Zone. Monthly trend from voyage_summary.AnchorTimeHours. v2 error corrected: v2 used Zone LIKE '%anch%' (lowercase only) which missed zones written with capital A (the actual MRTIS naming convention).

Cargo Size (MT)

Southport lineup mt_numeric field. Structural ~40--70% NULL rate expected. Non-null LOADING records used for size distribution. P25/Median/P75 via DuckDB PERCENTILE_CONT.

4. Data Caveats & Limitations

  • FGIS data covers Jan 1 -- Sep 25, 2025 only (Q4 extraction pending)
  • Southport lineup data: 255 of 365 possible 2025 report days captured
  • mile_ahp artifacts: COOPER DARROW, ZEN-NOH, LDC all show 164.0 -- true positions used from TERMINAL_META dict
  • Barge origin corridor percentages are structural USDA GTR estimates, not 2025 observed flow data
  • River stage values are historical climatological averages, not 2025 actual readings
  • Fog day count (34/year) is NOAA NWS New Orleans climatological average
  • ADM WOOD BUOYS and BUNGE DESTREHAN not in BWS GeoJSON -- approximate coordinates used
  • Stow factors computed from FGIS certified test weights (USDA official scale); not from vessel manifest or bill of lading
  • "Natchez Anch" in v2 fallback was fabricated -- removed in v3
